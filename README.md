# FitForge Wellness App

## Description
A C# console application developed to manage gym member details and evaluate eligibility for wellness rewards. The system processes member data, calculates fitness and activity metrics, and provides personalized nutrition suggestions.

## Tech Stack
* **Language:** C#
* **Framework:** .NET Framework 4.7.2
* **Environment:** Visual Studio

## Key Features
* **Member Management:** Captures and stores detailed member profiles, including join dates, age, and employment status.
* **Eligibility Assessment:** Applies complex business logic to determine wellness reward qualifications based on active membership duration (>24 months), treadmill distances, and monthly smoothie consumption rates.
* **Data Search & Statistics:** Includes functionality to search for members by name and generates percentage-based statistics on overall membership qualifications.
* **Dynamic Recommendations:** Generates weather-based smoothie suggestions utilizing local temperature inputs.
